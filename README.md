# RN Test Task

Expo Go project. Not compatible with bare / dev-client builds.

## Run

1. Install [Expo Go](https://expo.dev/go) on your phone (iOS App Store or Google Play).
2. Install dependencies:
   ```
   yarn install
   ```
3. Start the dev server:
   ```
   yarn start
   ```
4. Open the project in Expo Go:
   - **iOS** — scan the QR in the terminal with the Camera app.
   - **Android** — scan the QR from inside the Expo Go app.

Your phone and computer must be on the same network.
